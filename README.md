## 🐢 Turtle Race Game – Development Roadmap

10분 주기로 진행되는 거북이 레이스 게임입니다.  
웹(Next.js) 기반으로 개발하며, 이후 모바일 앱 확장을 목표로 합니다.

---

### 📅 Week 1 – Core Features (핵심 기능)

- 현재 레이스 조회 API  
  - 진행 중인 레이스 정보 반환  
  - 다음 레이스 시작 시간 제공

- 카운트다운 UI  
  - 다음 레이스까지 남은 시간 표시  
  - 실시간 업데이트

- 레이스 시뮬레이션 로직  
  - 거북이별 능력치 기반 레이스 진행  
  - 동일 조건에서 동일 결과 재현 가능

- 배팅 UI & API  
  - 거북이 선택 및 배팅 금액 입력  
  - 배팅 데이터 검증 및 저장

---

### 🎮 Week 2 – Gameplay & UX (게임 플레이)

- 거북이 애니메이션  
  - 레이스 진행에 따른 이동 애니메이션  
  - 결과와 연동된 시각적 연출

- 결과 표시  
  - 레이스 종료 후 순위 표시  
  - 승패 및 배당 결과 반영

- 배팅 내역  
  - 유저별 배팅 기록 조회  
  - 최근 레이스 결과 확인

- 간단한 유저 시스템  
  - 비회원 / 간단 유저 구분  
  - 기본 재화(머니) 관리

---

### ✨ Week 3 – Polish & Release Prep (다듬기 & 출시 준비)

- 통계 & 랭킹  
  - 승률, 누적 수익 등 기본 통계  
  - 유저 랭킹 보드

- 사운드 효과  
  - 레이스 시작 / 종료 / 승리 효과음  
  - 최소한의 몰입감 강화

- 모바일 최적화  
  - 모바일 화면 대응  
  - 터치 UI 최적화

- 앱스토어 준비  
  - PWA 또는 앱 패키징  
  - 스토어 등록용 빌드 및 메타데이터 정리

---
